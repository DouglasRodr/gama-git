As seguintes tecnologias/ferramentas foram usadas na construção do projeto:

- Storybook
- Styled Components
- Redux
- SweetAlert2

## 🚀 Executando Projeto

No diretório do projeto, você pode executar:
### `yarn start`

Para visualizar os stories no Storybook, você pode executar:
### `yarn run storybook`
